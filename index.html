支付行业概念浅析（一）

2013年08月16日 | Comments

支付行业近年来进入了一个战国时代，各种标准，行规，技术，概念，产皮都层出不穷，第三方支付，网银，POS，刷卡器，手机银行，快捷支付，IC卡等等都在不停的试水新方向，试图摸清或者是引导消费者支付习惯。

于是就导致概念错综复杂，不管是用户还是业内人士，都疲于应付新名词，却鲜有能对此行业做一番梳理的，一来是行业内缺乏有效的讨论和交流的环境，二来是没有国外可以照搬来参照的清晰体系。

本文试图澄清和解释一些基本的概念，希望能对理解此行业乱象有所帮助。

线上vs线下

以支付宝为代表的第三方支付，其实最初的需求是在C2C的电子商务模式中，提供一个可信媒介，来解决交易信任的难题。那是在网络和电子商务尚在起步阶段的时候。

在此之前，交易只有线下的时候，除了通行的现金之外，就只有POS–银行卡这种，由银行构建的，商家和消费者使用的线下支付体系。

电商的崛起改变了整个社会的消费习惯，也让第三方支付快速发展，C2C行业中，支付宝和支付通占据绝大部分市场，B2C电商也有将近30%的销售额来自线上支付，线上支付成为一块儿足够大的蛋糕，以支付宝为例，其日沉淀现金能达到200~300亿。

书读序言

2013年06月12日 | Comments

少年时候读书，是在毫无压力的环境下，因此不带丝毫功利性，全然是被书中有趣知识或者情节所吸引。那时候读书，基本上是随便翻开一页来读，觉得有趣的，便手不释卷的一路通读下去，直到末尾了才又回过头来将前面未读的给补上，有时甚至读到最后，根本不记得书名作者是什么，更不用说序言后记之类，无一例外被忽略。

现在想来，是因为那时候接触到的书，基本上都是父母专门挑选给我的，或者是长辈们的收藏，因此选书鉴别这个环节，从来不需要考虑，抄起一本就是不错的读物。

千禧年后，开始读高中，校园环境封闭压抑，此时书的来源变成了县城仅有几家小书店的盗版书和校外旧书摊。由于经费与时间都比较紧张，为了保证选到的书质量可靠，也有意无意的开始翻看序言，以求午饭到午休这段短短的时间内，可以从序言中管窥全书面貌，不至于买到一本读来无趣的书，白白浪费了午休被窝内的读书好时光。

现在回想起来，受限于当时的环境，阅读面非常狭窄，以各种盗版文学名著为主，古诗文和少量传记为辅，偶尔能从旧书摊淘来几本精彩的知识类书籍，那算是极大的收获了。

文学类书的序言，大致有这么两个路数，自序一般就是假惺惺的故作谦虚一番，故意轻描淡写提到受到某圈内著名作家谬赏不胜惶恐、假装不经心提到和某某知名评论家上次一起吃饭或者信中相互寒暄，是两种最常见的手法，当年我还年少无知没见过世面，经常被里面大牛的名字唬的一愣一愣，稀里糊涂中就把书买了回去。

而代人做序呢，一般就是文人见常见的相互吹捧，这点更为常见些，但是作序者通常只吹捧对方人品性格经历，很少有直接吹捧文笔和作品的，但凡遇到这种，需加倍留心，因为经验来看，这种序言的书，要么极精彩，要么就是极烂，对当时我不多的零花钱而言，更像是买了创业板股票，要承受更大的风险。

相对来说，我自己还是更喜欢作者自序多一些，我最喜欢的一些序言，基本都是作者的自序。毕竟作者自己更了解自己的书，从一些序言中还能了解到不少别处难以觅得的知识，当年读过徐志摩在《猛虎集》里面的序言，就让我长久以来在诗人们是怎么开始写诗的这件事儿上的好奇心得到了很难从别处得到的解答。

有的书序言甚至比书本身还要精彩，《史记》中的太史公自序中司马迁对儒墨法名道的评述和末尾“序略，以拾遗补，成一家之言，厥协六经异传，整齐百家杂语，藏之名山，副在京师，俟后世圣人君子。”这种范儿带给当时我的震撼超出书中任意一篇传记。

之前文章中提到的《宽容》一书的序言，出现在当时的课本里，也是篇极其精彩的序言，将一部历史书的主题用寓言的形式写进一则简单优雅的小故事中，这样的作者，你还需要怀疑他的睿智和文笔吗？假如说文如其人，这种情况下完全可以说成是“序如其书”了。

还有一篇序言不得不提，就是罗素在其自传中的自序，后来有个标题叫“我为何而活”，初次读到，感觉就如同当场被击中心脏，本来感觉心底若有若无有种模糊的东西你在追逐，此刻仿佛瞬间被点燃，变得光芒一片，这种喜悦和震撼，用“夫子言之，与我心有戚戚焉”也无法完全形容，文章中字句萦绕脑海，三日不绝，一点儿也不夸张，以至于后来我每次提及，恨不得全篇引用之。

前几年网上有段恶搞《无间道》的视频里一句曾志伟扮演的韩琛在和陈道明一起拜佛的时候的台词:”谁NB供谁”。这话很适合拿来描述大学时代读书的心境。彼时互联网上就有列书单的风气，诸如“计算机专业应该读什么”之类的小文非常受追捧，同学间也受这种风气影响，认为谁NB就读谁的书，就是书一定要读大牛的，仿佛谁读了书牛，读完了也能“立地成牛”一样。结果争相从图书馆借牛书，结果发现牛书中唯一看得懂的部分就是序言。这就导致当年同学间交流读书心得，变得像黄宏小品里拿名片打扑克一样，各自绞尽脑汁从读过的序言中搜刮一知半解的词句来证明自己读过的某本书比对方读过的更牛。

当年我虚荣心爆棚，几乎翻遍学院图书馆牛书，熟读其中半数的序言，吹起牛来文不加点，也算一时风光无二。

时至今日，为生活所迫，躬耕于一家名不见经传的小破IC设计公司，终日在各种狗屎代码之间艰辛的debug，桌上被一堆丑陋的电线占据，虚荣心是荡然无存了，但是心浮气躁，功利心在所难免。于是开始买各种技术书，看豆瓣各种书没看完就发表的不靠谱评论，买回来之后，迅速翻到所需要的章节，看完搞定工作后就丢在一边。技术类书籍的序言，大多朴实无华，大概分为两类，一类极其条理，描述这书讲了什么主题，讲到哪个层次，适合什么样的人看，作者都很谦逊并且实在，生怕读者被书名误导买入一本对自己无用的书，IT人务实的个性，可见一斑；另外一类，多是介绍写书和技术的背景，这类人物往往功底深厚，乃是对技术本身爱的深沉，以至于写书的时候都无法抽离这种感情。

印象深刻的有一本书是《C++ Primer》的作者大牛Lippman写的，书名叫《Essential C++》，作者就在序言中坦诚，写作本书是因为在自己学习Perl的时候，觉得自己被奉为C++入门经典的《C++ Primer》那本大厚书，实在是能让初学者望而生畏，令人沮丧，因此重写一本200来页的小书，以便真的将C++这门庞杂的技艺浅显而扼要的讲给入门者。当年的初学者如我，刚开始在大学时候被教材和庸师误导，满腔愤懑与疑惑，欲学而不得要领，读到此序此书，实属万幸。

但是网络时代读书，大家往往购自各大B2C书店，很多书是在买回来之后，才看到序言的，此时的序言已经不像当年那样对购买一本书起到什么关键或者辅助作用了。

这还算好的，其实互联网上盗版泛滥，大多数书都可以下载的到pdf，像此时的功利心，有时候读某书只是用的到其中一两个章节的内容，一时找借口为自己开脱下，down一份pdf下来，连版税都不用付给作者，此时便不大好意思去读作者的序言了，尤其是其中作者坦言乃工作之余牺牲私人生活时间来写作分享知识，末了还感谢家人谅解。在pdf上看到序言中这样的段落，实在是觉得良心有愧，难以对自己交代，索性不去翻看序言了。

近日来工作不忙，情绪不佳，不愿出门，只得在家翻翻旧书，通读是没那个心境，睡前挑着读两页，或者翻翻序言，能勾起对书中情节内容的回忆，也是一种乐趣，这感觉如同睹物思人，也像是听到老歌想到旧时朋友，温情与落寞杂糅并存。

昨天翻到《看上去很美》，看到王朔在序言《现在就开始回忆》中写自己如何自省早期作品为了趣味简化生活（歪曲生活），看他写多么迫切的想写真实，但是真实又是如何“一笔糊涂帐”而有“具有摧毁性”，还看到他写自己最终纠结到“大哭而回”，觉得连聪明透彻如王朔，也一样在纠结和自我否定中前行，于是心有宽慰，便不再抑郁，开始写此文。

理解欧几里得算法

2013年05月06日 | Comments

这篇文章翻译自Alice and Bob Cryptoland

Understanding the extended Euclidean algorithm

理解扩展欧几里得算法

The Euclidean algorithm finds the greatest common divisor of two numbers a and b. There is an extended version of it that also finds two numbers x and y such that ax + by = gcd(x,y). This is useful when searching for modular multiplicative inverses.

欧几里得算法寻找两个数字a和b的最大公约数。它有一个扩展的版本，同样可以用来寻找两个数字x和y，满足 ax + by = Gcd(x ,y). 这是一个很有用的算法，来寻找模运算的逆值。

The algorithm is simple, but I’ve never bothered to study why and how it works (a shame, really, but sometimes you have to postpone the understanding of some basic things in order to go on…). Finally I’ve decide to put some thought on it and came up with this (this is not a proof; it’s just some intuitive thinking to grasp the inner workings of the algorithm).

算法本身很简单，但是我从来没有为它是如何起作用或者为什么会起作用的事情所困扰（#没有去费力弄明白#）（这实在令人惭愧，但有时候你不得不跳过试图理解一些基础的东西，来继续你的工作）。最终，我决定花点时间思考它，然后带来这篇文章。（这不是一个证明，仅仅是一些直觉性的想法，去理解算法运行的机制）。

Suppose that you want to compute the extended Euclidean algorithm for a = 14 and b = 101. You can write

假设你要计算a = 14 ，b = 101的扩展欧几里得算法，你可以这么做：

14 *   1 + 101 *   0 =  14
14 *   0 + 101 *   1 = 101

Which is obviously true. Now subtract the first equation from the second:

这当然是对的。现在将第一个等式从第二个等式中减去：

14 *   1 + 101 *   0 =  14
14 *   0 + 101 *   1 - (14 *   1 + 101 *   0) = 101 - 14 ->
14 *  -1 + 101 *   1 = 87

You can repeat this many times before reducing the right-hand 101 in the second equation to the smallest positive value possible. Notice that it’s faster to take the floor(101/14) and subtract the first equation times the result from the second. In this case, we have floor(101/14) = 7. Therefore, replace the last step with:

你可以重复多次这个过程，直到将第二个等式右边101减到最小可能的有效值。注意这里计算（101/14）= 7. 因此，将最后一步替换为：

14 *   1 + 101 *   0 =  14
14 *   0 + 101 *   1 - [7 * (14 *   1 + 101 *   0)] = 101 - [7 * 14] ->
14 *  -7 + 101 *   1 =   3

Now, switch equations, in order to make the first one the one with the smallest right-hand size (you’ll always need to switch):

现在，交换等式两边，直到等式右边值最小为止（在此过程中你可能总是需要交换等式两边的值）：

14 *  -7 + 101 *   1 =   3
14 *   1 + 101 *   0 =  14

And here you are. Just repeat the same step over and over:

现在你得到这个结果，重复相同的步骤多次：

14 *  29 + 101 *  -4 =   2
14 *  -7 + 101 *   1 =   3

14 * -36 + 101 *   5 =   1
14 *  29 + 101 *  -4 =   2

14 * 101 + 101 * -14 =   0
14 * -36 + 101 *   5 =   1

When your first equation has right-hand side equal to 0 (this will always happen), the second equation will be ax + by = gcd(x,y)!

当你第一个等式的右边值等于零的时（这种情况总是会发生），那么第二个等式就是 ax + by = gcd(x,y)！

As I mentioned, this is useful in order to find modular inverses: to find which number that when multiplied by 14 gives 1 (modulo 101) you run the algorithm as described and finds that it’s -36 (which is 65 modulo 101). That is, 14 * 65 = 910, which when divided by 101, gives remainder 1. Modular inverses are used in many cryptographic applications, such as RSA.

像我提到的那样，这是对于求模逆值的非常有效的方法：去找到哪个数字乘以14然后（modulo 101）结果为1，你运行上面描述的算法，可以找到这个结果为-36（就是65 modulo 101）. 这样，14 * 65 = 910， 被101整除后余1.模逆在许多密码应用种都被用到，例如RSA.

I think I’ll study next the seemingly magical Montgomery reduction (I understand how it works, I have even implemented it, but I still don’t know why the heck it works)…

我想我接下来会学习#改进的蒙哥马利约减算法#（我理解它如何起作用，我也曾经使用过它，但是我还是不知道为什么它可以起作用）。

《冰与火之歌》，一部非同寻常的巨著

2013年05月04日 | Comments

我向来对科幻小说，奇幻文学之类的东西不太感兴趣。大概是大部分作品所谓的想象力不怎么能打动我，对着很多明显是咬着笔尖硬挤出来的那么点生硬的所谓“xx世界观”和七拼八凑的幼稚的“想象力”无法忍受超过三页纸的限度。然而这本书却不在其中。

乔治.R.R.马丁的《冰与火之歌》是一部气势恢宏的奇幻巨著，作者在其中构建了一个极其庞大而完整的世界，有着类似于中世纪欧洲的社会结构，有着这个世界特有的历史、文化、宗教和传说，有着数以百计的性格各异而又命运交织的人物形象，以及荡气回肠的英雄主义悲剧情节。

之所以说是一部巨著，是因为作者一开始就把写作的目标定为像托尔金《魔戒》一般的大部头三部曲形式。但是作者老马丁在写作的过程中觉得三卷的篇幅无法完整的展开他的故事，而后一再扩充构想中的篇幅，目前作者开列了七卷的写作计划，目前已经有五卷出版。

（早期的旧版四卷有中译本，我当初看到的也应该是这个旧版本。目前好像第五卷中译本刚刚出来，我之前断断续续在网上看到的第五卷都是网友们自己挑了某些章节翻译的。朋友Fred刚刚买了第五卷并热情借阅与我，我打算抽出时间来完整的看看。在此特别感谢Fred同学。）

虽然作者声称作品构思受到托尔金的影响，但是本书和《魔戒》之间的差异还是显而易见的。

读魔戒的时候主要是被书中描绘的庞大而精致的场景所震撼，但是魔戒本身却是一个单线的，很传统的故事，虽然有紧张的情节，但是读起来不会有太大的压力。总体来讲你知道虽然魔都的索隆和那些黑骑士都是些恐怖压抑的形象，但是托尔金的小说中主角明确的是一个热爱生活，善良友好贪吃的小霍比特人，有一堆朋友的帮助，就连咕噜都故意写得那么好玩，带有一种讲英国人特有的，围坐壁炉边故事的腔调。

两相比较，《魔戒》本质上是一个老少皆宜的霍比特人历险记故事，《冰与火之歌》则是一部有着令人望而生畏篇幅的英雄主义悲情史诗。

虽然这可能会影响一部分人的热情，但我还是要说，阅读这本书绝不是一件轻松的事情。

首先是作者在其中使用了在我看来比较新颖的POV（Point of view）的叙述方式，每一个章节都由一个人物的主视角来展开。因此本书的故事线是由无数个叙事线索交织而成的，有时候齐头并进，有时候却又相互重复纠错，因此读者在阅读的时候需要适应这种不断变换角度和价值观的叙事方式，以及这种叙事方式带来的比常规叙事多出许多来的信息量。这些主视角人物，不仅有各个家族的关键角色，也有底层人物，仆从，侍卫，不同的人物有着不同的背景，使命，从不同的角度解读正在成为维斯特洛大陆历史的一个个事件。

其次是作者野心十足的构思了庞大的故事架构。幅员辽阔，让《魔戒》中的中土大陆看上去像飞行棋游戏棋盘的维斯特洛大陆，有着悠久的神话传说和历史，有着复杂的内政外交，类似于中世纪一般运行着的的社会结构，同时还带有一点点神秘色彩的魔法元素。加上前文提到的POV叙事手法，让作者可以轻松的在庞大的历史感叙事和人物心理微小的细节之间转换自如。但是对于读者来说，刚开始阅读起来的感受却是，很难或者根本无法把握故事的脉络，经常发生读者的预期被作者的构思远远抛下的挫败感。更形象一点来说，就是读者往往被POV叙事带进一个人物的视角，并由此产生了代入感，但是随着故事的展开，却发现这只是其中的一个小分支，读者心中的“主角”会在历史大背景下毫无预兆的被抛弃，然后随着阅读的深入，会发现，之前阅读的小说中的简单的主角光环在这部书中根本不存在。

这引出了我想谈的关于这部小说的另外一点，就是写实。我认为将《冰与火之歌》归为奇幻小说实在是有点贬低本书了。虽然整个故事的背景都是虚构，维斯特洛世界中也有龙和魔法出现，但是这些元素只是一些艺术化的手法，让故事带有某种神秘色彩的调调的必要元素（实际上我认为优秀的小说都不会以这些东西作为主要卖点）。书中既有精彩的历险，也有惊心动魄的宫廷斗争，既有复杂阴险的政治，也有恢宏惨烈的战争，同时也不惜笔墨的塑造了无数个性格各异的人物，但是作者并未单独凸现某一个事件某一个人物的重要性，而是冷静的将人物和事件置于维斯特洛的环境与历史之中，任其在无情的环境中拼命挣扎，与命运的漩涡抗争，做出艰难的抉择，付出惨烈的代价。所有的一切，不管是魔幻的还是艺术的，不管是理想的还是现实的，不管是阴谋的还是荣耀的，都会在情节的推动下坍塌，让人感觉作者似乎在有意强调历史的残酷冰冷无情。

最后，与《魔戒》明媚欢快的节奏不同，《冰与火之歌》开篇从绝境长城和临冬城起笔就奠定了一种悲壮苍凉的基调。POV叙事的代入感和英雄主义悲剧的基调，让书中性格身世各异的人物的坎坷命运成为了让读者最无法释卷的原因。正式作者着力强调的个人的抗争在历史车轮下的无力感，凸显了人物命运的悲剧性。无论是为荣誉而死的临冬城主奈德.史塔克，还是背负弑君者骂名的詹米.兰尼斯特，甚至连龙石岛的洋葱骑士身上，处处都有着历史对命运特有的一种不仁慈，令读者动容。

此外，阅读本书的额外收获是其中的警句谚语。举几个例子：

“最愚蠢的人通常也比嘲笑他们的家伙聪明。”

“死的历史用墨水书写，活的历史则用鲜血。”

（Dead history is writ in ink, the living sort in blood.）

“恐惧比利剑更伤人。”

（Fear cuts deeper than swords.）

“在权力的游戏之中，你不当赢家，就只有死路一条，没有中间地带。”

“在权利的游戏之中，最卑微的棋子也有自己的欲望，有时候会拒绝执行你为它们设计的行动。”

警句的英文原文往往更有韵律感，本文中就不一一附上原文了，还有一句有意思的警句是用小说中虚构出来的瓦雷利亚语说的，叫“凡人皆有一死”，和太史公的那句著名的话一样。另外几大家族都有自己的格言家训，其中我觉得以临冬城（winterfell）史塔克家族的最有感觉。winter is coming，冷风吹起，凛冬将至，冷峻感十足。

在阅读这本书的过程中我常常会被作者肆无忌惮的展开每一个本以为不起眼的人物疯狂扩张篇幅，让故事线以惊人的速度发散的做法而吓到，据说作者已经是70高龄了，让人不禁担心马丁老先生能否在生前不留悬念的以7卷的篇幅成功将故事收尾，后来断断续续阅读第五卷看到作者有收敛苗头的时候终于稍稍缓了一口气。

以上只是我试图从不同角度来介绍《冰与火之歌》这本书的特别之处所做出的尝试，事实上对于这么庞大的一部小说而言，上面这些介绍可能仍属于片面，加上我第五卷还没有完整的读过，难免有主观臆断的猜测，但是可以肯定的是这是一部足够精彩的小说，值得一试。

《宽容》,另一种角度的人类历史

2013年03月31日 | Comments

亨德里克.威廉.房龙作品

我偏爱历史类的书籍。

英国作家佛朗西斯.培根在文章《谈读书》里面有句话说“读史使人明智”。我想他说的是历史，绝不仅仅是我们在学校里学到的那些一长串的年代表，也不是一个个单独的隔离的事件，更不是供人消遣的小故事。容我简单粗暴的来形容一下，历史在房龙眼里，至少是一个人类文明的完整标本，包含了无数可供后人研究借鉴的东西，你从不同的角度来看，就能得到不同的结论和感受。

从这个角度来讲，房龙绝对算得上是一个明智的人，他用独特的文笔和浅显的叙事语调在几十本著作中从不同的角度差不多将西方历史给读者解读了个遍，而我最喜欢的，是这本序言曾经出现在中学课本里的《宽容》(无知谷寓言小故事，还有印象吗？)。

作者在这本书里面，挑选了一个非常有趣的角度。我想这是我第一次见到有人从宽容的角度来讲历史。在这本书里面，历史可不像标本一般枯燥无味，而是由一位对西方历史和基督教历史了如指掌，颇有造诣的人，掩卷沉思之后，将自己思考的心得，用讲与朋友分享的语调写就，极其引人入胜又发人深省。我对此书的喜爱，一是源于作者这些思考，另外一部分就是这种让人读起来很舒服的语调了。

需要提及的是，书中大量的篇幅都在讲基督教相关的信仰宽容。我认为这也是了解基督教的一个不错的主线，阅读这书也解答了我很多关于宗教的疑惑。但是作者也提到，宗教自由其实仅仅是一个开端，人们摆脱掉宗教的不宽容之后，还有种族的不宽容，社会的不宽容，以及许多其他人尚未意识到的，但是却根深蒂固的不宽容。

书中有几点读后觉得比较震撼，其中一点是，人类社会的进步是曲折而迂回的，每一次用巨大牺牲争取来的自由与进步，稍不留意，人们就会陷入之前他们反抗的对立面上去，自大而顽固，亲手毁掉来之不易的成果。异见者往往在他们成为主流意识的时代里加倍镇压将来的异见者。作者在其中忧心忡忡的写道“宽容就如同自由，只是乞求是得不到的，只有永远保持警惕才能保住它”，对于“革命”作者提醒说“历史的奇怪法则之一就是表面变化越大，就越一成不变”。

作者在其中也多次肯定了知识和同情的作用（这也是我另外一个喜欢的作家罗素在自己自传序言中提到的），知识可以用来化解无知带来的恐惧，同情心则是人类善良的本源。总体来说，作者还是持有一个乐观的态度，甚至对人类的不宽容也持有一个宽容的态度，认为人类“毕竟很年轻”，虽然有很长很曲折的路要走，但是最终宽容这种美德也是可以通过人类对进步的追求获得。

信息技术进步给人类活动带来了新的变革，对于生活在互联网时代的我这一代年轻人来说，在获取知识和表达感受前所未有的容易的今天，我觉得这本书尤其有意义。

每周一书

2013年03月26日 | Comments

写这个主要是为了和大家多交流一些读书的心得。

近年来疲于应付工作生活，读书量每况愈下。偏偏现在可供阅读的书籍和获得书的渠道都太容易，导致面临选择问题，如何花时间来选一本好书，真是件难事。

所以想到这个主意，读书接龙，每周推荐一本自己读过的书给朋友，下周轮换另外一位朋友推荐，既保证有书可读，又把寻书的重任分担下去，目前来看，还是一个不错的方案。

跟大家提起这个想法，还是挺受欢迎，至于实施起来效果如何，倒是可以慢慢尝试。大家一起参与，也能集思广益，不断改进。

望今年能多读几本好书，多认识几位朋友。
